# /sort-documents

Ziel: Projektbezogene Dokumente sauber sortieren.

## Standardordner

```text
docs/Dokumente/
├── 00_INBOX/
├── 01_BRIEFINGS/
├── 02_RESEARCH/
├── 03_ASSETS/
└── 04_ARCHIV/
```

## Ablauf

1. Prüfe `docs/Dokumente/00_INBOX/`.
2. Erstelle zuerst nur einen Sortiervorschlag als Tabelle:
   - Datei
   - vorgeschlagener Zielordner
   - Begründung
3. Verschiebe nichts ohne Michels Bestätigung.
4. Nach Bestätigung verschiebe die Dateien.
5. Aktualisiere danach `docs/PROJECT_STATUS.md`.

## Zielordner

`01_BRIEFINGS/`:
- Übergaben
- Anforderungen
- Projektbeschreibungen
- ChatGPT-Handouts
- Kunden-/Nutzerbriefings

`02_RESEARCH/`:
- Recherchen
- Marktanalysen
- Wettbewerber
- technische Vergleiche
- Quellen

`03_ASSETS/`:
- Bilder
- Logos
- Screenshots
- Mockups
- Beispiel-Dateien
- Designmaterial

`04_ARCHIV/`:
- alte Versionen
- überholte Dokumente
- verworfene Varianten

Wenn unklar: in `00_INBOX/` lassen und Rückfrage stellen.
