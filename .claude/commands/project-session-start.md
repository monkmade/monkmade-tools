# /project-session-start

Ziel: Eine aktive Projekt-Session sauber starten.

## Ablauf

1. Prüfe den aktuellen Projektordner.
2. Lies, falls vorhanden:
   - `README.md`
   - `CLAUDE.md`
   - `docs/PROJECT_STATUS.md`
3. Fasse kurz zusammen:
   - Was ist dieses Projekt?
   - Was ist der aktuelle Stand?
   - Was ist der nächste sichere Schritt?
   - Welche No-Gos gelten?
4. Nichts ändern.
5. Frage Michel, womit gestartet werden soll.

Wenn `docs/PROJECT_STATUS.md` fehlt, schlage vor, sie anzulegen.

Wenn `CLAUDE.md` fehlt, schlage vor, sie anzulegen.
